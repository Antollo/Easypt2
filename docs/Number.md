## Name: `Number`

#### Instance of: [`Class`](Class.md)

### Description:

Represents number represented as at least 4 bytes
integer or at least 8 bytes floating-point number,
implementation should choose an appropriate representation
dynamically.

#### Example:

```js
console.write(1 + 2 * 3);
```

##### Expected output:

```
7
```
