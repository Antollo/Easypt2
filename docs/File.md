## Name: `File`

#### Instance of: [`Class`](Class.md)

### Description:

File class provides file input and output operations.

#### Example:

```js
console.write("1+3 is ", 1+3);
if (1+3 != 4) {
    console.warn("1+3 is not 4!");
}
```

##### Expected output:

```
1+3 is 4
```
