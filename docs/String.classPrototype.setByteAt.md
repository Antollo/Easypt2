## Name: `String.classPrototype.setByteAt`

#### Instance of: [`Function`](Function.md)

#### Parameters: `position /*Number*/, byte /*Number*/`

### Description:

Sets new value of character (byte at given position).

#### Example:

```js
let x = "abc";
x.setByteAt(0, 98);
console.write(x);
```

##### Expected output:

```
bbc
```
