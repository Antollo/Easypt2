## Name: `String.classPrototype.erase`

#### Instance of: [`Function`](Function.md)

#### Parameters: `position /*Number*/, [length = string.length() /*Number*/]`

### Description:

Erases bytes from range [position, position + length).

#### Example:

```js
let str = "abcdefg";
str.erase(2, 3);
console.write(str);
```

##### Expected output:

```
abfg
```
