## Name: `Object.classPrototype.readOperator`

#### Instance of: [`Function`](Function.md)

#### Parameters: `key /*String*/`

### Description:

Read operator (string[key]) returns property.

#### Example:

```js
let obj;
obj.number = 21;
obj["secondNumber"] = 22;
console.write(obj["number"], " ", obj.secondNumber);
```

##### Expected output:

```
21 22
```
