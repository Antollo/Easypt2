## Name: `Array.readOperator`

#### Instance of: [`Function`](Function.md)

#### Parameters: `...`

#### Return value: `/*Array*/`

### Description:

Static read operator ([1, "A"]) constructs new Array.
Not to be confused with method Array.classPrototype.readOperator.

#### Example:

```js
let arr = [36.6, "Harry", true];
console.write(arr[1]);
```

##### Expected output:

```
Harry
```
