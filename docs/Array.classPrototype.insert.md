## Name: `Array.classPrototype.insert`

#### Instance of: [`Function`](Function.md)

#### Parameters: `n /*Number*/, element`

#### Return value: `this`

### Description:

Inserts new element before n-th element.

#### Example:

```js
let arr = [36.6, "Harry", true];
arr.insert(1, "Dirty ");
console.write(arr[1], arr[2]);
```

##### Expected output:

```
Dirty Harry
```
