## Name: `Array.classPrototype.length`

#### Instance of: [`Function`](Function.md)

#### Return value: `/*Number*/`

### Description:

Returns the number of elements in that array.

#### Example:

```js
let arr = [36.6, "Harry", true];
console.write(arr.length());
```

##### Expected output:

```
3
```
